How to Create a Scene

Follow these steps to create a scene and control multiple devices with a single tap.
  1. Click on Scenes in Footer
  2. Click on Create Scene
  3. Select the Devices you want to include in your scene
  4. Click Configure for all the switches you have selected
  5. You can click Edit Configuration and set the switch status again for particular switches and click OK
  6. Click OK to confirm all configurations
  7. Toggle Auto Activate for a routine and set a time you want to activate the scene
  8. Toggle Manually to create a scene that you can operate whenever you want

Device-Specific Settings

  • If you have selected an RGB strip, set brightness and color to your convenience
  • If you have selected a fan, set speed to your convenience
  • Click OK after configuring each device
Note:
  • You can edit configurations at any time by selecting the scene and clicking Edit
  • Auto Activate scenes will run at the scheduled time every day
  • You can create multiple scenes for different activities and occasions
Tip:
  • Create scenes for common activities like "Movie Night", "Dinner Time", or "Reading"
  • Use Auto Activate for scenes you want to run at fixed times, like "Good Morning" or "Bedtime"
  • Give each scene a descriptive name that makes it easy to identify
