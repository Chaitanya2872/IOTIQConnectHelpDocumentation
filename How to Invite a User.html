Manage Projects: How to Invite a User

Follow these steps to invite a new user to your project.
  1. Click on MenuBar
  2. Click on Manage Projects
  3. In Your Projects, click Invite Users
  4. Click Add New Invitation
  5. Select the type of User (Secondary User or Guest User)

Invitation Code

  • A 12-digit code is generated automatically
  • Share this code with the user you want to invite
  • The code format looks like: ULYN-F6WF-FMLU
  • You can copy the code and view the type of user you have selected
  • Each invitation code is valid for one user only
Important:
  • One invitation code can be used by only one user.
  • Create separate invitation codes for each user you want to invite.
Note:
  • You can delete the invitation at any time by clicking on Delete Invitation in Manage Invitations Screen
  • Deleting an invitation will not remove access for users who have already joined.
