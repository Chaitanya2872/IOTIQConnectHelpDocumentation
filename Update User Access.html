Manage Access

Follow these steps to update access permissions for Secondary or Guest Users.
  1. Click on Menu Bar
  2. Click on Manage Projects
  3. Click on Manage Users
  4. Click on Manage Access
  5. Toggle on the Secondary User or Guest User based on the type of User you want to update
  6. You can also change spaces, rooms and devices for which guest user was initially assigned
  7. Click on Update User Access
Note:
  • The access changes will be immediately reflected in their project view.
Removing a User
  • You can remove a user by clicking on Remove User instead of Manage Access
  • You can add the user again at any time by generating a new invitation code
Tip:

It's good practice to review user access permissions periodically to ensure they align with current project needs.
