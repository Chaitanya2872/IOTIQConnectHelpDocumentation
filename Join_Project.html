Join Project

Follow these steps to invite a new user to your project.
  1. Click on MenuBar
  2. Click on Manage Projects
  3. In Footer, click Join Project
  4. Enter 12-digit Invitation code shared by the primary user
  5. Click on Join
  6. Select the Joined Project and Click Change to access that project

Invitation Code

  • Each invitation code is 12 digits in format:
  • The code format looks like: ULYN-F6WF-FMLU
  • Only primary users can generate invitation codes.
  • The code expires immediately when its used.
Important:
  • Invitation Code will be Expired once its used.
  • When you've joined another project as a secondary or guest user, you can't generate invitations on behalf of the primary user
Note:
  • You can Manage Multiple Projects by joining through Invitation Codes
  • You can leave anytime from the project by clicking on Leave Project in the Manage Projects screen
  • Leaving a project will remove your access but will not delete any data you contributed.
