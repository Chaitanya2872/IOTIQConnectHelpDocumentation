Add a Room

Follow these steps to add a new room to your space.
  1. You can Add a Room when a Space is Created
  2. Click on Add New Room in your created space
  3. Select Room Type (e.g., Living, Bedroom, Dining, Lawn, etc.)
  4. Add a Room Name
  5. Select Floor and Click on Continue

Alternative Methods to Add Rooms

  • You can also add rooms through Manage Rooms Screen
  • You can also add a room by clicking on AddRoom+ beside the room label
Note:
  • Creation of Room allows you to Add Devices and control them
  • Each room can have its own unique name and type
  • Rooms can be organized by floors for multi-level spaces
Tip:
  • Use descriptive room names for easy identification
  • Organize rooms by floors to match your physical space layout
  • Select the appropriate room type to access relevant device templates
